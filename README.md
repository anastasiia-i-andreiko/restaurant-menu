# restaurant-menu

Консольний застосунок для управління меню ресторану.
Розроблений в рамках командної роботи (1 учасник з роздвоєнням особистоті).

# Функціонал

-  Додавання страв з назвою, описом, ціною та категорією
-  Редагування страв (ціна, опис, категорія)
-  Видалення страв за назвою або категорією
-  Підрахунок загальної ціни та ціни по категоріях
-  Перегляд меню по категоріях
-  Сортування страв за ціною

# Запуск

```bash
python menu.py
```

# Структура гілок

- `main` — базова версія
- `dev` — основна гілка розробки
- `feature/A-add-dish` — додавання страв (А)
- `feature/B-edit` — редагування (Б)
- `feature/C-delete` — видалення (В)
- `feature/D-price` — ціна і сортування (Г)
